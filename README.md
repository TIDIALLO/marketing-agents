# Synap6ia Marketing Agents

Plateforme d'automatisation marketing pilotée par IA — 3 agents spécialisés orchestrés via n8n.

## Agents

| Agent | Nom | Mission |
|-------|-----|---------|
| Agent 1 | Content Flywheel | Création → Adaptation → Publication → Tracking → Signaux gagnants |
| Agent 2 | Amplification Engine | Veille concurrentielle → Créatives pub → Campagnes ads → Scaling |
| Agent 3 | Opportunity Hunter | Ingestion leads → Qualification AI → Nurturing → Booking d'appels |

## Stack

Next.js 16 · Express.js 4 · PostgreSQL 16 · Redis 7 · n8n · Claude API · OpenAI (DALL-E/Whisper)

## Méthode

Développement structuré via **BMAD Method** v6.0.0-Beta.7 (Breakthrough Method of Agile AI Driven Development).
